# UECS2453---Food-Delivery-Job-Sequencing-Problem
2026 March @ UTAR 2453 Problem Solving with Data Structures and Algorithms (Assignment Task: Job Sequencing Problem)

## Task: To find a sequence of jobs, which will be completed within their deadline constraints and provides maximum profit
This repository contains a job sequencing problem where the topic is decided on Food Delivery Tasks, similar to the known industry of GrabFood. This is to evaluate and compare the efficiency of different algorithms that will be present in this system by using Big O notation. These algorithms are implemented in Java and utilizing the object-oriented programming approach to ensure the program is flexible.
